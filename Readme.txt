Разработка программы учета товаров, хранящихся на складах.
Основные поля: товар (название, ед. измерения, количество), список товаров.
Основные функции: ведение списка товаров, определения общей стоимости товаров на складе, 
прием/отпуск товаров, поиск товара по различным признакам, ведение справочника единиц измерения.
Написано на С.
